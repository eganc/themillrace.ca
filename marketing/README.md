# The Millrace Co. - Oracle Interface

This is the codebase for the marketing/landing page of **The Millrace Co.**, featuring the "Oracle" AI interface.

**Live Site:** [themillrace.ca](https://themillrace.ca)

## Documentation
- **[Deployment Guide](./DEPLOY.md)**: Instructions for Vercel & Cloudflare.
- **[Development Log](./DEVELOPMENT_LOG.md)**: How to update the site and use branches (Safe Mode).

## Setup
1.  Navigate to `marketing`: `cd marketing`
2.  Install dependencies: `npm install`
3.  Run locally: `npm run dev`

## Tech Stack
- React + Vite
- Google Generative AI SDK
- Custom Canvas Particle Engine
